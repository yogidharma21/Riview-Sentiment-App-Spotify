# Riview-Sentiment-App-Spotify
This project is a Sentiment Analysis application for Indonesian text, powered by a fine-tuned IndoBERT model.
Tech Stack
* Tensorflow
* Pytorch
* Transformers
* IndoBERT
* Python 3
* Streamlit
* Hugging Face
